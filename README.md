Healthcare Robot SLAM (ROS 2 + RealSense + ESP)
This project is a real-world ROS 2 based healthcare delivery robot. The robot uses an Intel RealSense depth camera for SLAM (mapping and localization) and an ESP-based motor controller for movement. The goal is to autonomously navigate in environments like hospitals and deliver medicines.

What this project does
Builds a map of the environment using SLAM
Localizes the robot inside the saved map
Navigates to goal locations
Sends velocity commands to an ESP motor controller
Visualizes everything in RViz
